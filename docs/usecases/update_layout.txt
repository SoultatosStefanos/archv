Update Layout

Data:
	Graph, Components
	Topology
	Type

Primary Course:
	1. User issues "Update Layout" command with the layout type data.
	2. System validates layout type.
	3. System returns if the layout type is the same as the current.
	4. System creates a new layout, given the type and the above cached data.
	5. System spreads the graph, given it new layout.

Exception Course: LayoutType Error
	1. System delivers error message to user.
